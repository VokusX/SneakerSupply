
******* LOCAL INSTRUCTIONS ********

Launch your MongoDB Server using:
   $ mongod

In the root folder for the Assignment, in the /seed folder, run the following
   $ node populate-for-startup.js

Modify the API keys for PayPal in the app.js file to your own.

Then go back to the root folder, and run:
   $ npm start

You can then access the local server running at http://localhost:3000
